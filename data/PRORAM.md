# Полная программа курса

## Общая рамка

Программа курса на [LLM Start](https://llmstart.ru/fullstack-aidd/) покрывает полный цикл AI-driven fullstack-разработки:

- работа с LLM и AI-инструментами
- backend и API
- проектирование и интеграция базы данных
- frontend-разработка
- DevOps и автоматизация
- production deployment

## М01. Основы LLM и AI-кодинг экосистема

### Описание

Введение в современные LLM, AI-first подход и экосистему инструментов для AI-driven разработки.

### Теория

- Принципы работы и возможности современных LLM
- Промпт-инжиниринг и контекст-инжиниринг
- Концепция ИИ-агентов и AI-first подход в разработке
- Обзор моделей и провайдеров
- Эволюция подходов к AI-кодингу
- Консольные агенты: Claude Code, Codex, Warp и другие
- IDE-интегрированные решения: Cursor, Windsurf и другие
- Инструменты для быстрого прототипирования: V0, Bolt, Lovable, Replit
- Обзор дорожной карты практики курса

### Практика

- Создание скрипта автоматизации с помощью AI-инструментов
- Создание UI-прототипа в одном из AI-сервисов прототипирования

- Конфигурирование моделей, правил и соглашений
- Управление контекстом
- Лучшие практики использования кодовых агентов
- Генерация скелета приложения и структуры проекта
- Реализация базовой бизнес-логики приложения
- Интеграция с LLM API
- Управление состоянием и контекстом
- Обработка ошибок и логирование

### Практика

- Настройка Cursor под проект
- Разработка стартовой версии AI-driven приложения (Telegram-бота) с интеграцией LLM
