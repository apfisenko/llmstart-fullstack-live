1. Название проекта

Telegram-бот для сопровождения учебного потока курса AI-driven fullstack developer

2. Краткое описание идеи

Бот помогает участникам учебного курса ориентироваться в материале, отвечает на вопросы по программе и поддерживает процесс обучения в рамках курса AI-driven fullstack developer.

3. Для кого бот

Для слушателей курса, которые проходят учебный поток в рамках курса AI-driven fullstack developer.

4. Какую задачу решает

Помогает быстро получить ответы на вопросы по содержанию и организации курса, облегчает навигацию по учебному материалу.

5. Что умеет в MVP

- Вести диалог в Telegram
- Отвечать на вопросы в рамках своей роли, заданной системным промптом
- Помогать с навигацией по курсу
- Давать простые ответы на вопросы по содержанию курса (на основе README.md и PROGRAM.md)
- Давать ответы на основе своей параметрической памяти

6. Примеры вопросов к боту

- Где найти расписание занятий?
- Какой следующий модуль по программе?
- В каком документе описан домашний проект?
- Что включает в себя второй модуль?
- К кому обратиться по организационным вопросам?
