## Сделано

- Прокси `GET` к backend: [`frontend/web/app/api/v1/cohorts/[cohort_id]/leaderboard/route.ts`](../../../../../../../frontend/web/app/api/v1/cohorts/[cohort_id]/leaderboard/route.ts)
- Типы ответа: [`frontend/web/lib/leaderboard-types.ts`](../../../../../../../frontend/web/lib/leaderboard-types.ts)
