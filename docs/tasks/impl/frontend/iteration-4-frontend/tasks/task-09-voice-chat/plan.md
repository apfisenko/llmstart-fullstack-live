# Задача 09: голосовой режим чата — решение

**Статус:** ⛔ **Отменена** (не реализуется).

## Контекст

По [tasklist-frontend.md](../../../../../tasklist-frontend.md) задача предполагала голосовой ввод/вывод для веб-клиента и Telegram-бота с сохранением принципа «ядро — backend».

## Решение

После анализа и планирования принято решение **не выполнять** задачу 9: сочетание сценариев (браузерные Speech API, права, совместимость; для бота — загрузка voice, STT на backend или у провайдера, лимиты, новые контракты и интеграции) даёт **неприемлемое техническое усложнение** относительно ценности на текущем этапе.

## Следствия

- Код и API под голос **не добавляются** в рамках этой задачи.
- Повторный подъём темы — отдельное решение с оценкой roadmap (итерации 5–6), возможным ADR и новой задачей в tasklist.
